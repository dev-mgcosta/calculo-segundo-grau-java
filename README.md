# Cálculo de Equação do Segundo Grau em Java

Projeto acadêmico desenvolvido durante a faculdade para praticar lógica de programação em Java.

Estou adicionando este projeto ao GitHub agora com o objetivo de organizar meu histórico de estudos e registrar minha evolução na programação.

## Sobre o projeto

O programa solicita os valores de a, b e c, calcula o delta e exibe as raízes reais da equação do segundo grau quando elas existem.

## Conceitos utilizados

Java  
Lógica de programação  
Entrada e saída de dados  
Estruturas condicionais  
Laço de repetição
